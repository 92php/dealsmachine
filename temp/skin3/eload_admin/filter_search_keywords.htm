搜索关键字过滤设置
不参与搜索关键字:
以半角逗号隔开，不支持词组，如cell phone
如果以这些关键字开始的就不入库:
如果以这些关键字结束就不入库:
搜索排除对应关系:
一行一个，格式：关键字 => 排除搜索关键字。如phone=>iphone,speakerphone,headphone。不支持词组，如cell phone
单复数&同义词处理:
一行一组相近词，不同词用英文逗号分开，eg: 'cell phone,cell phones,mobile,mobiles' ,搜索 'mobile',相当于同时搜索
cell phone
cell phones
mobile
mobiles
